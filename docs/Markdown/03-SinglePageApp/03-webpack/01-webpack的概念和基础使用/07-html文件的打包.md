**2 笔记**
* HTML的生成  
    **需要用到的plugin**
    > * `npm install html-webpack-plugin --save-dev`  

    **相关配置**  
    > * `filename`：打包生成的html文件的名字  
    > * `template`：指定一个html文件为模板  
    > * `minify`：压缩html  
    > * `inject`：是否把js，css文件插入到html，插入到哪  
    > * `chunks`：多入口时，指定引入的chunks
