**2 笔记**
* 什么是前端模块化  
    **非模块化的不足**
    > * 协同开发不方便，一不小心就污染了别人的代码  
    > * JavaScript之间无法相互引用  
    > * 不方便分配工作  
    > * 引入别的模块用script，需要特别注意顺序

    **为什么要模块化**
    > * 前端工程日益复杂，必须多人协作，模块化更适应多人协作  
    > * 模块独立，方便后期维护
